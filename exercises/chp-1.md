**10 다음 ( ) 안에 적당한 용어를 채워 완성하시오.**

1. (데이터)는 현실 세계에서 관찰 도는 수집의 결과로 냐타난 사실(fact) 이다.

4. (데이터 일관성)은 중복되거나 연관된 데이터 간의 의미적 불일치성 혹은 모순성이 없는 상태를 말한다.
5. (데이터 정확성)은 저장된 데이터 값이 현실 세계의 실제 값과 얼마나 정확히 일치하는지를 뜻하는 의미적 정확성을 말한다.

**11 데이터와 정보의 차이를 구별하여 설명하시오.**
데이터는 현실세계에서 관찰 또는 수집의 결과로 나타난 객관적 사실이고 정보는 획득된 데이터에 의미를 부여하거나 가공 처리를 통해 의사결정에 활용하도록 체계적으로 조직한 결과물입니다.
데이터는 익히지 않아 먹지 못하는 소고기이고 정보는 적절히 구워서 먹기 좋은 스테이크라고 생각합니다.
**12 정보 시스템의 구조와 종류를 설명하시오.**
정보시스템은 한 조직의 활동과 운영에 필요한 데이터를 수집, 저장해두었다가 다양한 방식으로 처리 및 가공함으로써 의사결정에 필요한 정보를 생성하는 소프트웨어 체계를 의미합니다.
종류로는 생산정보시스템, 마케팅정보시스템, 인사정보시스템, 재무회계정보시스템, TPS, MIS, ES, DSS, ERP등이 있습니다.

**13 파일 정보 시스템 방식과 비교할 때 데이터베이스 시스템 방식이 갖는 장점은 무엇인지 설명하시오.**
데이터베이스 시스템은 파일 정보 시스템 방식보다 데이터 종속성이 최소화되고, 데이터 중복성이 감소하며, 데이터 동시공유가 가능하고, 뎅터 일관성 유지가 쉬워지고, 데이터 무결성 유지가 쉽고, 데이터 보안이 향상되고, 표준화된 데이터 접근이 가능하고, 데이터 가용성이 향상되고, 응용프로그램의 개발비용이 감소합니다.

**14 시대별 데이터베이스 시스템의 성장 과정을 개략적으로 설명하시오.**
1970년대 정보화 이전 시대에는 별도의 정보시스템이 없었습니다.
1980년대 종보화 초기에 컴퓨터를 도입하여 파일 안에 데이터들을 저장하며 파일 정보시스템이 등장했습니다.
1990년대 데이터베이스를 구축하고 DBMS가 설치된 서버를 운영함으로써 누구나 인터넷을 통해 통합된 데이터베이스 안에 데이터를 저장하거나 접근하게 되었습니다.
2000년대 웹 데이터베이스와 데이터 활용의 고도화를 위한 데이터웨어하우스 등이 데이터베이스 시스템에 도입되었습니다.
2010년대 데이터 저장소의 확장성과 가용성이 강조되면서 NOSQL 데이터베이스 기술 등이 활용되기 시작했습니다.

**15 좁은 의미의 데이터베아스를 정의하시오.**
좁은 의미의 데이터베이스는 데이터베이스를 보다 효율적이고 체계적으로 활용하기 위해 데이터 형식이나 구성에 엄격한 제약을 갖는 시스템 관점에서의 데이터베이스를 의미합니다.

**16 데이터베아스의 주요 특성 4가지를 설명하시오.**
1. 많은 사용자들이 다양한 목적을 위해 공동으로 소유하고 유지하는 공용 데이터
2. 여러곳에서 필요한 데이터를 하나로 통합한 통합 데이터
3. 컴퓨터를 통해 직접 접근이 가능한 0과 1의 이진 문자열로 표현되는 디지털화된 저장 데이터
4. 특정 조직의 운영 목적을 위해 사용되는 운영 데이터
